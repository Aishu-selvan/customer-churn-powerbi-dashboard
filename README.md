# Customer Churn Analysis & Risk Segmentation Dashboard (Power BI)

## Project Overview
This project presents an interactive **Power BI dashboard** to analyze customer churn patterns and identify high-risk customers. The dashboard helps understand how customer engagement, subscription plans, and behavioral patterns influence churn probability.

The objective is to provide **data-driven insights** that help businesses improve customer retention and reduce churn risk.

---

## Business Problem
Customer churn is a major challenge for subscription-based companies. Identifying customers who are likely to leave helps organizations take proactive actions to improve customer satisfaction and loyalty.

This dashboard analyzes customer data to:
- Identify high-risk customers
- Understand factors affecting churn
- Analyze customer behavior patterns
- Support decision-making using insights

---

## Tools & Technologies Used

- Power Query (Data Cleaning)

---

## Dashboard Features

### KPI Metrics
- Total Customers
- Churn Rate (%)
- High Risk Customers
- Average Churn Score

### Customer Segmentation
Customers are categorized into:
- High Risk
- Medium Risk
- Low Risk

### Behavioral Analysis
Analyzes relationship between:
- Daily watch time
- Customer engagement
- Churn risk level

### Subscription Plan Analysis
Compares churn score across:
- Basic Plan
- Standard Plan
- Premium Plan

### Device Type Analysis
Analyzes churn score by device:
- Desktop
- Laptop
- Tablet
- Mobile
- Smart TV

### Interactive Filters
Dashboard allows filtering by:
- Region
- Subscription Plan
- Customer segment

---

## Key Insights
- Customers with lower engagement tend to have higher churn risk
- Basic subscription plan users show slightly higher churn score
- A large number of customers fall under high-risk category
- Customer behavior insights can help improve retention strategies
